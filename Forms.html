O atributo type="text" está especificando o tipo de Valor que aquele imput receberá


Type="Button" cria um botão simples que é atibuto necessário para um evento, no qual o usuário pode clicar. Não é necessário criar esse imput dentro de um form.


Type="Checkbox é ultilizado para que o usuário escolha mais de uma opção


Type="email" ultilizado para que o usuário possa entrar com dados cujo valor seja um e-mail


Type="file" é ultilizado para que o usuário possa fazer upload de arquivos
